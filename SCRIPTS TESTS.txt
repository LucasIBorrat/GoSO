Comando para cambiar las IPs de todos los archivos de CONFIG (cambiar las IP 192.168.X.Y dependiendo el valor que nos de IFCONFIG)

sed -i 's/"IP_CPU": *"127\.0\.0\.1"/"IP_CPU": "192.168.0.127"/g' *.json
sed -i 's/"IP_MEMORIA": *"127\.0\.0\.1"/"IP_MEMORIA": "192.168.0.190"/g' *.json
sed -i 's/"IP_KERNEL": *"127\.0\.0\.1"/"IP_KERNEL": "192.168.0.107"/g' *.json
sed -i 's/"IP_IO": *"127\.0\.0\.1"/"IP_IO": "192.168.0.107"/g' *.json
